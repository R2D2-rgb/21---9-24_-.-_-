## 1. Описание проекта
Данный проект представляет собой прототип информационной системы (ИС) для автоматизации [указать цель: учета продаж, регистрации читателей и т.д.]. 
Система позволяет структурировать хранение данных и быстро получать аналитическую информацию через SQL-запросы.

##  2. Техническое задание (ТЗ) вкратце
*   **Цель:** Перейти от бумажного учета к автоматизированной базе данных на SQLite.
*   **Основные функции:**
    1. Регистрация новых записей (клиенты, товары, услуги).
    2. Поиск данных по заданным критериям.
    3. Формирование итоговых отчетов за период.
*   **Требования к БД:** Минимум 4 связанные таблицы, соблюдение целостности данных.

##  4. Реализованные SQL-запросы
В рамках проекта подготовлены сценарии для:
*   Подсчета общего количества [товаров/услуг].
*   Выборки данных с использованием `JOIN` (объединение таблиц).
*   Фильтрации данных по условию (например, цена > 1000).

##  Структура репозитория
*   `/data` — файл базы данных `project.db`.
*   `/sql` — скрипты создания таблиц (`create.sql`) и тестовые данные (`insert.sql`).
*   `/diagrams` — графическая схема базы данных.
*   `queries.sql` — файл с примерами сложных запросов.
