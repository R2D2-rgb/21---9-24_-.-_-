## 1. Описание проекта
Данный проект представляет собой прототип информационной системы (ИС) для автоматизации [цель: обеспечивать эффективное взаимодействие с клиентами, учёта их счетов, обработки транзакций и управления кредитными продуктами]. 
Система позволяет структурировать хранение данных и быстро получать аналитическую информацию через SQL-запросы.

##  2. Техническое задание (ТЗ) вкратце
*   **Цель:** Перейти от бумажного учета к автоматизированной базе данных на SQLite.
*   **Основные функции:**
    1.  Регистрация клиентов и счетов.
    2. Поиск транзакций по дате/сумме.
    3. Аналитика по кредитным продуктам.
*   **Требования к БД:** Минимум 4 связанные таблицы, соблюдение целостности данных.

##  3. Структура базы данных (ER-схема)
Проект базируется на следующих сущностях:
1. **[таблица 1 - Клиенты]** - (данные о клиентах).
2. **[таблица 2 - Счета]** - (номера счетов и балансы).
3. **[таблица 3 - Транзакции]** - (история операций).
4. **[таблица 4 - Кредиты]** - (информация о займах).

*Скриншот ER-диаграммы находится в папке `/diagrams`.*

##  4. Реализованные SQL-запросы
* Подсчёт активных клиентов.
* Выборка транзакций за месяц.
* Фильтрация кредитов с просрочкой.

1. Установите **sqliteviewer.app**.

3. Откройте файл из папки `/data` и выполните запросы из файла `queries.sql`.
