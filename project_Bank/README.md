# 🏦 Банковская система управления

## 📋 О проекте

Консольное приложение для автоматизации банковских операций, разработанное в рамках курсового проекта. Система позволяет управлять клиентами, счетами, транзакциями и кредитами.

### 🎯 Цели проекта
- Автоматизация основных банковских операций
- Ведение учета клиентов и их счетов
- Обработка транзакций с проверкой баланса
- Управление кредитным портфелем
- Формирование финансовой отчетности

## 🛠 Технологический стек

- **Язык программирования:** Python 3.8+
- **База данных:** SQLite 3
- **Библиотеки:** sqlite3 (стандартная библиотека)
- **Система контроля версий:** Git / GitHub
- **Среда разработки:** PyCharm / VS Code
